Import Guide

              
Use the strings exactly as-is.

              
    
                
  1. Open Codex Appearance settings.
    2. 
                
  2. Choose the Light or Dark import field.
    3. 
                
  3. Paste the copied string beginning with codex-theme-v1:.
    4. 
                
  4. Click Import theme. Codex applies it immediately even if it does not appear as a named preset.
    5. 
                
  5. Set fonts, cursor style, and font sizes separately in Appearance.
    6. 
              

            


            

              
Notes
